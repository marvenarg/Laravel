# Laravel Projects

## Español

### Descripción

Repositorio personal donde reúno distintos proyectos desarrollados con **Laravel**.

Aquí voy subiendo aplicaciones, trabajos prácticos, pruebas y proyectos realizados con diferentes funcionalidades del framework, incluyendo autenticación, CRUD, validaciones, APIs, testing, logging, exportaciones y otras herramientas del ecosistema Laravel.

El objetivo de este repositorio es centralizar ejemplos y desarrollos realizados con Laravel, manteniendo cada proyecto organizado de forma independiente.

### Contenido del repositorio

Este repositorio puede incluir proyectos como:

* aplicaciones web con autenticación
* CRUDs con relaciones entre entidades
* APIs REST con Laravel Sanctum
* proyectos con Laravel Breeze y Blade
* pruebas automatizadas con PHPUnit
* exportación de datos en distintos formatos
* trabajos prácticos y proyectos finales realizados con Laravel

Cada carpeta o proyecto puede contar con su propio `README.md` con instrucciones específicas de instalación y uso.

### Tecnologías y herramientas utilizadas

Según el proyecto, se pueden utilizar tecnologías como:

* Laravel
* PHP
* MySQL / MariaDB
* Laravel Breeze
* Blade
* Tailwind CSS
* Laravel Sail
* Vite
* PHPUnit
* Laravel Excel
* Sanctum

### Estructura

Cada proyecto se encuentra organizado en su propia carpeta, con sus archivos, migraciones, modelos, controladores, vistas, pruebas y documentación correspondiente.

Ejemplo general:

```text id="4jvkz3"
proyecto-1/
proyecto-2/
proyecto-3/
```

### Ejecución

Cada proyecto puede tener requisitos y pasos de instalación diferentes.

Por eso, para ejecutar cualquiera de ellos, se recomienda ingresar a la carpeta correspondiente y leer su `README.md`.

En general, los proyectos pueden estar preparados para funcionar en alguno de estos entornos:

* **Laravel Sail**
* **XAMPP / entorno local**
* **PHP + Composer + Node.js**

### Nota

Este repositorio funciona como colección de proyectos y prácticas hechas con Laravel.
La configuración, dependencias y modo de ejecución pueden variar entre carpetas.
